Example Images

The following images should be added to this directory:

  • product-before.jpg - A product photo (e.g., shoe, toy, gadget)
  • product-after.jpg - 3D model render of the same product
  • architecture-before.jpg - A building or interior photo
  • architecture-after.jpg - 3D model render of the same building
  • nature-before.jpg - A landscape or nature scene
  • nature-after.jpg - 3D model render of the same scene

Images should be 800x600px or similar aspect ratio for best results.
